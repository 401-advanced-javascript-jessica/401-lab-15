### Setup
#### `.env` requirements
* `PORT` - 3000
* `MONGODB_URI` - `mongodb://localhost:27017/lab15`

#### Running the app
* `npm start`
* Endpoint: `/api/v1/categories`
  * Returns a JSON object category data in it.
* Endpoint: `/api/v1/products`
  * Returns a JSON object with product data in it.
